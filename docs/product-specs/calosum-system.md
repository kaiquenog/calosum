# Calosum System

## Resumo

Calosum e um agente neuro-simbolico com:

- hemisferio direito para percepcao e estado latente;
- corpo caloso para compressao em soft prompts;
- hemisferio esquerdo para raciocinio tipado e acoes primitivas;
- memoria dual com consolidacao semantica;
- metacognicao com avaliacao de variantes.

## Requisitos Estruturais

- orquestracao assíncrona
- interfaces desacopladas
- persistencia opcional
- telemetria rastreavel
- fallback de runtime
