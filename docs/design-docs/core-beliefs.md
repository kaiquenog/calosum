# Core Beliefs

- O repositorio e a fonte de verdade para humanos e agentes.
- Contexto deve ser progressivamente revelado, nao despejado em um arquivo gigante.
- Fronteiras importam mais do que estilo local.
- Regras recorrentes devem virar codigo, teste ou lint.
- Debt deve ser pago continuamente para evitar propagacao de padroes ruins.
